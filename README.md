## caela-uk/os • Installation Script

A simple, beautiful and kinda user-friendly **GNOME and Flatpak/Flathub installation script**

This script installs GNOME as bare bones as possible with a clean wallpaper then proceeds to install all of my favourite apps and utilities, preferring Flatpak wherever possible. It also includes some nice tweaks like adw-gtk3-dark to theme old gtk3 apps like the included GParted and configures a nice wallaper and some basic GNOME/GTK settings.

This script is designed to run on Arch right after a clean installation with no DE or apps but _may possibly_ work on other basic Arch-based distributions with no confirmation.
